# Add Authenticated App Shell

## What to build

Create the authenticated web shell and backend auth seam so a Google-authenticated user can be represented consistently in the app.

## Acceptance criteria

- [ ] The web app has a dashboard entry point.
- [ ] Backend exposes current-user and auth-sync routes.
- [ ] Tests can use a fake authenticated user without depending on Google.
