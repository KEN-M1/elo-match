# Create and View a League

## What to build

Allow an authenticated owner to create a 1v1 Elo league and view its leaderboard shell.

## Acceptance criteria

- [x] A league can be created with name, slug, description, and public visibility.
- [x] The creator becomes an admin member with the starting rating.
- [x] The league detail view shows members ordered by rating.
