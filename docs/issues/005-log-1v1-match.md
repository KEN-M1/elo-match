# Log a 1v1 Match

## What to build

Allow a league member to report a 1v1 match against another member.

## Acceptance criteria

- [ ] A match can be logged with winner, loser, reporter, and played date.
- [ ] Logged matches start as pending confirmation.
- [ ] No ratings change while a match is pending.
